# **App Name**: Tech Solutions Hub

## Core Features:

- Hardware Product Display: Display a grid of available hardware products on the 'Laptops', 'Desktops', and 'Accessories' pages. Include key details like name, image, and price, as well as an 'Add to Cart' button.
- Detailed Product View: Show a comprehensive view of the selected product, including high-resolution images, technical specifications, and customer support resources.
- User Authentication: Implement user registration and login using Firebase Authentication.  Ensure form validation and clear error handling, with redirection upon successful login/registration.
- Contact Form: Include a 'Contact Us' page with a submission form for user inquiries.  Store form data in Firestore.
- Technical Support Access: Allow customers to browse available IT professionals (dummy data) on a dedicated page, view their profiles, and submit support requests.
- AI Chatbot Integration: Incorporate a chatbot as a customer support tool on the website.  The chatbot will provide immediate assistance and guidance, powered by a third-party AI API, to answer common questions.
- Dark Mode Toggle: Allow the user to choose a dark or light color theme for the application, and persist the theme selection to local storage.

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) for a modern and tech-oriented feel.
- Background color: Light gray (#E0E0E0) for a clean, professional look.
- Accent color: Soft purple (#B15DFF) to highlight interactive elements.
- Body font: 'Inter' (sans-serif) for a modern, readable style.
- Headline font: 'Space Grotesk' (sans-serif) for a clean, tech-focused header.
- Use crisp, modern icons that represent functionality, sourced from a consistent icon set (e.g., Material Design Icons).
- Employ a grid-based layout for product listings, ensuring responsiveness across devices.
